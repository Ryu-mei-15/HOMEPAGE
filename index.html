About

Projects

CatcherX — VR Catcher Training

Unity / XR Interaction / 物理シミュレーション / 2025-

投球軌道の物理再現、捕球動作入力、触覚・視覚フィードバックを備えた捕手体験。球種・球速・回転をパラメタ化し、緊張感のある実装を追求。

UnityVRPhysicsHaptics

LuVoX — Real‑Time MIDI → Visuals

MIDI / GLSL / Python Librosa（準備） / 2025-

入力MIDIに反応して映像を合成するパフォーマンス志向のシステム。オンセット解析や鍵盤生成ロジックとも連携予定。

MIDIReal‑timeVisual

Marble Animation — Onset‑Synced Keys

Python / Pygame / Librosa / 2024-

音楽のonset_attackを元に、スフィアと鍵盤の生成・接触タイミングを制御。外部JSONから正確な時刻列を読み取り、視覚表現を発展。

PythonLibrosaPygame

Baseball Analytics & VR Visualization

R / Python / Unity / 2023-

投球のリリース〜捕球時間、マグヌス力・回転ベクトル、球種別到達時間のモデリングと可視化。VR訓練系との統合を志向。

RUnityData Viz
